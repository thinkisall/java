-Memory

1) stack = <메소드 호출시> 지역변수, 매개변수, 리턴값등의 데이터를 임시저장하는데 사용.
이후 메소드가 끝나면 해당 데이터는 스택에서 제거

2) heap = <인스턴스 객체들 생성시> 객체를 저장하는 공간.

객체생성 => heap에 공간할당 => 생성자 호출 => 객체 초기화 => 객체 heap메모리에 할당 => 객체에 대한 참조 변수를 통해 접근가능
관리는 GC를 통해서 관리되어진다.
