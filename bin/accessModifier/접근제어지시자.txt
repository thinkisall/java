** 접근 제어 지시자 **

클래스 외부에서 클래스의 멤버변수, 메서드, 생성자를 사용할 수 있는지 여부를 지정하는 <키워드>

1) private = 같은 클래스 내부에서만 접근가능 (외부 클래스, 상속 불가)

2) 아무것도 안적을때 = default 

=> 같은 패키지 내부에서만 접근 가능 ( 상속 관계라도 패키지가 다르면 접근불가)

3) protected = 같은 패키지나 상속관계의 클래스에서 접근 가능

4) public = 클래스의 외부 어디에서나 사용가능
